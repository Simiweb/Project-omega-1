Software

Software sind Programme, mit denen man Computer betreibt. Dank Software passiert überhaupt etwas auf dem Computer. Sie enthält Regeln und Befehle, damit der Computer weiß, was er tun soll. Das Gegenstück zur Software ist die Hardware: Das sind die Teile des Computers, die man anfassen kann.

Server

Es braucht für sehr viele Dinge einen Server. Zum Beispiel ist diese Webseite auch auf einem Server, sodass jeder darauf zugreifen kann. Unser Computer oder unser Handy kommuniziert viel mit Servern. Einen Server braucht man zum Beispel um im Internet etwas zu googeln oder ein Game zu spielen aber auch Apps sind irgendwo auf einem Server gespeichert! Wenn eine Webseite langsam ist, oder nicht reagiert, kann es sein, dass ein Server überlastet ist oder einen Error hat.
